# Requirements

- Node 18.16.0

# Installation

- Clone this repository

  ```bash
  git clone https://github.com/enriquemolinari/curso-react-modulo1.git
  ```

- Go to the project folder

  ```bash
  cd curso-react-modulo1
  ```

- Install dependencies

  ```bash
  npm install
  ```

- Run the project
  ```bash
  npm run dev
  ```
